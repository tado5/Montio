================================================================================
🔥 TROUBLESHOOTING - ČASTÉ PROBLÉMY
================================================================================

PROBLÉM: Backend sa nespustí
RIEŠENIE:
  pm2 logs montio-backend --err
  Check .env súbor
  Check DB connection

PROBLÉM: CORS errors vo frontende
RIEŠENIE:
  nano /var/www/montio/backend/.env
  Pridajte: CORS_ORIGIN=https://yourdomain.com
  pm2 restart montio-backend

PROBLÉM: Rate limiting nefunguje
RIEŠENIE:
  ls -la backend/middleware/rateLimiter.js (musí existovať)
  grep "rateLimiter" backend/routes/auth.js (musí byť importovaný)
  pm2 restart montio-backend

PROBLÉM: 500 errors
RIEŠENIE:
  pm2 logs montio-backend --err --lines 100
  Často: missing .env variable alebo missing DB column
  Re-run migration: mysql -u root -p montio_db < database/migrations/simple_migration_v1.11.0.sql

PROBLÉM: Upload fails
RIEŠENIE:
  mkdir -p backend/uploads/avatars backend/uploads/logos
  chmod 755 backend/uploads backend/uploads/avatars backend/uploads/logos
  sudo chown -R www-data:www-data backend/uploads/
